# Скрипты и руководство по установке Arch Linux.

Этот репозиторий содержит сценарии установки, файлы конфигурации и документацию, которые я использую для развёртывания и настройки Arch Linux.

## Основные компоенты

### Базовая система

* Arch Linux
* Linux Zen
* Unified Kernel Image (UKI)

### Сеть

* systemd-networkd
* systemd-resolved
* iwd

### Загрузчик

* Limine (опционально)
* Secure Boot через sbctl (опционально)

### Дополнительное ПО

* PipeWire
* Fish
* Neovim
* UFW
* NVIDIA Open Driver (опционально)
* Intel Undervolt (опционально)

## Установка

Перед началом необходимо:

* загрузиться с официального установочного образа Arch Linux;
* подключиться к Интернету;
* подготовить разделы диска (ESP, корневой раздел и, при необходимости, раздел подкачки).

> Подробное руководство по подготовке и установке будет добавлено позже.

Скачайте и запустите `install.sh`:

```bash
curl -fsSLO https://git.postmodernist.ru/Rabbit/etc/raw/branch/master/install.sh
```

После первой загрузки выполните:

```bash
~/etc/post-install.sh
```

`post-install.sh` выполняет завершающую настройку системы и позволяет установить дополнительное программное обеспечение. Часть компонентов являются необязательными и могут быть пропущены.

## Предупреждение

Скрипты изменяют разделы диска, устанавливают пакеты и заменяют системную конфигурацию.

Перед использованием рекомендуется ознакомиться с их содержимым и запускать их только в том случае, если вы понимаете, что именно они делают.
